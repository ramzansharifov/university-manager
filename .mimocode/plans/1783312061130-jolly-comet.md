# University Structure Refactor Plan

## Goal
Refactor from `Faculties ŌåÆ Departments ŌåÆ Specialties ŌåÆ Groups` to `Faculties ŌåÆ Specialties ŌåÆ Groups` with departments as a separate entity linked to faculties via many-to-many.

---

## Phase 1: Database Schema

### 1a. Update `001_initial_schema.sql`
- Remove `faculty_id NOT NULL` from `departments`, make it nullable or remove
- Add `applies_to_all_faculties INTEGER NOT NULL DEFAULT 0` to `departments`
- Create `department_faculties` junction table
- Remove `department_id NOT NULL` from `specialties`
- Change `specialties` unique constraint from `(department_id, name)` to `(faculty_id, name)`
- Update indexes

### 1b. Create migration `008_refactor_departments_and_specialties.sql`
- Create `department_faculties` table
- Migrate existing `departments.faculty_id` ŌåÆ `department_faculties` rows
- Add `departments.applies_to_all_faculties`
- Rebuild `specialties` table without `department_id` NOT NULL constraint (safe copy)
- Preserve all data

### 1c. Update `schemaRepair.ts`
- Ensure `department_faculties` table exists
- Ensure `departments.applies_to_all_faculties` column exists
- Migrate legacy `departments.faculty_id` ŌåÆ `department_faculties` if column exists
- Handle legacy `specialties.department_id` safely

### 1d. Update `schemaValidation.ts`
- Validate `department_faculties` table and its columns
- Validate `departments.applies_to_all_faculties`
- Keep validating `specialties.faculty_id`

## Phase 2: Backend

### 2a. Update `adminCrud.ts` types
- Add `'department_faculties'` to `AdminEntityKey`

### 2b. Update `adminCrudEntities.ts`
- **departments**: Remove `faculty_id` from editable, add `applies_to_all_faculties`
- **specialties**: Remove `department_id` from editable
- **department_faculties**: Add new entity config

### 2c. Update `adminCrudService.ts`
- **`prepareDepartmentData`**: Remove `faculty_id` requirement. Validate head/deputy as before. Add `applies_to_all_faculties` handling.
- **`prepareFacultyData`**: Rewrite `ensureTeacherBelongsToFaculty` to check via `department_faculties` or `applies_to_all_faculties=1`
- **`prepareStudentGroupData`**: Curator validation path becomes `specialty ŌåÆ faculty ŌåÆ department_faculties/applies_to_all ŌåÆ teachers`
- **`ensureTeacherBelongsToFaculty`**: Query `department_faculties` + `applies_to_all_faculties` instead of `departments.faculty_id`

### 2d. Update Data Quality
- `dataQualityRepository.ts`: Rewrite `countFacultiesWithoutDepartments` to use junction table. Replace `countDepartmentsWithoutSpecialties` with `countDepartmentsWithoutFacultyScope`.
- `dataQualityService.ts`: Update titles/descriptions for changed checks.
